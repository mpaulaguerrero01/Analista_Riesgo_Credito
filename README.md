# Analista Riesgo Crédito

Este repositorio contiene el análisis financiero de diferentes emisores del sistema financiero colombiano con el objetivo de evaluar la conveniencia de incluir uno de ellos dentro de un portafolio de inversión.

El análisis se basa en información pública de la Superintendencia Financiera de Colombia e incluye la construcción de indicadores clave de tamaño, crecimiento, rentabilidad, riesgo crediticio y estructura financiera.

A partir de estos indicadores se elaboran visualizaciones comparativas y un scorecard de evaluación que permite identificar el perfil riesgo–retorno de cada emisor y sustentar la recomendación final de inversión.
